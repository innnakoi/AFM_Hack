# AI Shield Guardian Frontend

React-based dashboard for real-time threat monitoring and analysis.

## Features

- Real-time system metrics (CPU, Memory, Processes)
- Threat detection visualization
- Process monitoring and management
- Network connection tracking
- Threat history and alerts
- Explainable AI threat analysis

## Setup

```bash
npm install
npm start
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
